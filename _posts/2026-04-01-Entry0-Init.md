Hi, this is a sort-of-first entry of my journey-ish-thing.

As a first step, I set up my laptop with an SSH key pairing.
I'm not really sure how it works, but with a little bit of playing around and looking up how to set it up I've managed to create an ssh key pairing and my personal lenovo laptop is now fully authenticated and linked!

I created a repo with git in WSL via powershell, and added the standard hello world txt. Next step was to create a github repo with commands like gh repo create. The biggest surprise was how easy and comprehensible the steps were that github built into the command!

The only thing that confused me was its request to create a remote. I had really no clue what that meant, but with some searching I assume it means that creating a remote means theres now a link between my local repo and the repo on github. If I hadn't done this, the repos would be fully seperate from eachother (?). Also didn't help there was no default value for yes or no, but alas, we input Y.

Now there should be a private repo online in which I'm going to build the blog as described in "Niels' bonustraject".
